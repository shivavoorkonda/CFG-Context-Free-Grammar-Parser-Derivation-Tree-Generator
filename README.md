# CFG Parser Simulator

A Web-based Context-Free Grammar (CFG) Parser and Simulator built with Python and Flask. This project allows users to input context-free grammars, process them, and test whether various strings belong to the language generated by the grammar using the Cocke-Younger-Kasami (CYK) algorithm.

## Features

- **Grammar Parsing:** Input your Context-Free Grammar in a simple text format.
- **Epsilon Productions:** Supports epsilon ('e' or 'E') productions correctly.
- **CYK Algorithm Simulator:** Tests if a given string is accepted by the Context-Free Grammar using the dynamic programming CYK algorithm.
- **Step-by-Step Visualization:** Visualizes the CYK parsing table and string acceptance results in a dynamic web interface.
- **Error Handling:** Gracefully handles syntax errors, invalid rules, and rejected strings, providing clear feedback.

## Project Structure

- `app.py`: The main Flask backend application that handles routing and HTTP requests.
- `cyk.py`: Contains the core implementation of the Cocke-Younger-Kasami (CYK) algorithm for testing string membership.
- `cfg_parser.py`: Houses the logic for parsing, validating, and potentially normalizing the input context-free grammar.
- `templates/`: Contains HTML templates for the web interface (`index.html`).
- `static/`: Contains static assets like stylesheets (`styles.css`) for the frontend.
- `requirements.txt`: Project Python dependencies.

## Prerequisites

- Python 3.x
- pip (Python package installer)

## Installation and Setup

1. **Clone the repository or download the source code.**
2. **Navigate to the project directory:**
   ```bash
   cd path/to/Toc
   ```
3. **Install the dependencies:**
   It is recommended to use a virtual environment.
   ```bash
   pip install -r requirements.txt
   ```
   *(This will install Flask and any other required libraries.)*

## Running the Application

1. Start the Flask development server:
   ```bash
   python app.py
   ```
   *or equivalently:*
   ```bash
   flask run
   ```
2. Open your web browser and navigate to the local server address, typically:
   ```
   http://127.0.0.1:5000/
   ```

## Usage

1. **Enter Grammar:** Once the web application is loaded, enter your CFG rules into the provided text area. The format generally expects standard grammar notation (e.g., `S -> AB`, `A -> a`). Epsilon can be denoted with `e` or `E`.
2. **Test String:** Enter a string you'd like to test against the grammar.
3. **Parse:** Submit the form to run the CYK algorithm.
4. **Results:** The application will display whether the string is accepted or rejected, along with the generated CYK table for visualization.
